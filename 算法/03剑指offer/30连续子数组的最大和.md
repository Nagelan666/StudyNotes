# 连续子数组的最大和 

### 信息卡片 

- 时间： 2019-1-26
- 题目描述：

```
Z偶尔会拿些专业问题来忽悠那些非计算机专业的同学。今天测试组开完会后,他又发话了:在古老的一维模式识别中,常常需要计算连续子向量的最大和,当向量全为正数的时候,问题很好解决。但是,如果向量中包含负数,是否应该包含某个负数,并期望旁边的正数会弥补它呢？例如:{6,-3,-2,7,-15,1,2,2},连续子向量的最大和为8(从第0个开始,到第3个为止)。给一个数组，返回它的最大连续子序列的和，你会不会被他忽悠住？(子向量的长度至少是1)
```

 

### 参考答案

> 思路

本质就是返回连续最大子序列的和

sum：如果当前元素小于0，那么sum加上和只会变小，所以让sum直接等于当前元素，并让max记录当前获取到的最大值。

如果当前元素大于0，就让sum加上val，max记录下当前元素最大值。



**max的作用就是记录下扫描过程中的最大值，而扫描过程本质就是当val<0,sum=val，val>=0,sum就加上。**




> 代码

```java
import java.util.*;
public class Solution {
    public int FindGreatestSumOfSubArray(int[] array) {
        int sum = 0,max = Integer.MIN_VALUE;
        for (int i = 0; i < array.length; i++) {
            int val = array[i];
            if(sum < 0){
               sum = val;
            }else{
                sum += val;
            }
            max = Math.max(sum,max);
        }
        return max;
    }
}
```
