# 数字在排序数组中出现的次数 

### 信息卡片 

- 时间： 2019-1-27
- 题目描述：

```
统计一个数字在排序数组中出现的次数。
```



### 参考答案

> 思路

二分法查找。




> 代码

```java
public class Solution {
    int count = 0;
    public int GetNumberOfK(int[] array, int k) {
        if(array == null ||array.length == 0) return 0;
        int l = 0, r = array.length - 1;
        return bSearch(array, l, r, k);

    }

    private int bSearch(int[] array, int l, int r, int k) {
        int mid = (l + r) / 2;
        if (l <= r) {
            if (array[mid] == k) {
                count++;
                bSearch(array, mid + 1, r, k);
                bSearch(array, l, mid - 1, k);
                return count;
            } else if (array[mid] > k) {
                return bSearch(array, l, mid - 1, k);
            } else {
                return bSearch(array, mid+1,r, k);
            }
        }

        return 0;
    }
}
```
