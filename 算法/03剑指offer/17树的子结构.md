# 树的子结构 

### 信息卡片 

- 时间： 2019-1-23
- 题目描述：

```
输入两棵二叉树A，B，判断B是不是A的子结构。（ps：我们约定空树不是任意一个树的子结构）
```



### 参考答案

> 思路

递归。思路在注释里。




> 代码

```java
    public boolean HasSubtree(TreeNode root1,TreeNode root2) {
        if(root1 == null || root2 == null){
            return false;
        }
        //isSubtreeWithRoot(root1, root2) ：两棵树的根节点向下逐个进行比较
        //HasSubtree(root1.left, root2)：递归，以大树的左节点为根节点的树里面寻找小树
        //HasSubtree(root1.right, root2)：递归，以大树的右节点为根节点的树里面寻找小树
        return help(root1,root2) || HasSubtree(root1.left,root2) || HasSubtree(root1.right,root2);
    }

    private boolean help(TreeNode root1, TreeNode root2) {
        //注意root2和root1判断的顺序不能换哦
        if(root2 == null) return true;
        if(root1 == null) return false;
        if(root1.val != root2.val){
            return false;
        }
        return help(root1.left,root2.left) && help(root1.right,root2.right);
    }
```
