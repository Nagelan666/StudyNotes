# 二维数组中的查找 

### 信息卡片 

- 时间： 2019-1-21
- 题目描述：

```
在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。
```



### 参考答案

二分法

- 以左上角对角线为搜索的基准，先进行行搜索，再进行列搜索

- 图解

  ![1](./assets/1.png)

  

```js
public class Solution {
    public boolean Find(int target, int [][] array) {
            if(array == null || array.length == 0){
                return false;
            }

            int width = array[0].length;
            int height = array.length;
            int min = Math.min(array.length,array[0].length);
            for (int i = 0; i < min; i++) {
                boolean flag  = binarySearch(array,target,i,width - 1,true)
                                ||  binarySearch(array,target,i,height - 1,false);
                if(flag){
                    return true;
                }
            }

            return false;
        }

    private boolean binarySearch(int[][] matrix, int target, int l, int r, boolean flag) {
        int start = l;
            //搜索横排
            if(flag){
                while(l <= r){
                    int mid = (l + r) / 2;
                    if(matrix[start][mid] < target){
                        l = mid + 1;
                    }else if(matrix[start][mid] > target){
                        r = mid - 1;
                    }else{
                        return true;
                    }
                }
            }else{
                while(l <= r){
                    int mid = (l + r) / 2;
                    if(matrix[mid][start] < target){
                        l = mid + 1;
                    }else if(matrix[mid][start] > target){
                        r = mid - 1;
                    }else{
                        return true;
                    }
                }
            }
             return false;
            }

}
```
